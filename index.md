### Welcome to Machine Learning Tutorials.
Run K-Nearest Neighbors on the famous MNIST dataset instantly:

```
$ git clone git@github.com:lazyprogrammer/machine_learning_examples.git
$ cd machine_learning_examples
$ python knn.py
```
